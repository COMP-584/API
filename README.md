# Chat App API

## Running for Developement
> npm run dev

## Running for testing
> npm run test

## Running for production
> npm start

## Running the API server

1. Install dependencies
```bash
npm i
```

2. Create `/config/.env` file with...
```txt
PORT=<port number>
MONGO_URI=<Mongo DB URI>
JWT_SECRET=<secret password>
```

3. Run the server
```bash
npm start
```
